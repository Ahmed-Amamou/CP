# Competitive Programming Solutions

This repository contains solutions to various competitive programming problems from different platforms.

## Repository Structure

- **practice/** - Practice files, templates, and utilities for local testing
- **Codeforces/** - Solutions to Codeforces problems
- **UsacoGuide/** - Solutions to USACO Guide problems
- **CSES 150 Challenge/** - Solutions to CSES problemset
- **Project Euler/** - Solutions to Project Euler problems
- **OnlineJudge/** - Solutions from various online judges
- **Advent of code 2024/** - Advent of Code 2024 solutions
- **Advent of code 2025/** - Advent of Code 2025 solutions
- **Contests/** - Contest participation solutions
- **Notes/** - Notes and references

## Getting Started

The `practice/` folder contains useful templates and utilities:
- Debug template for C++
- Input/output templates for Python
- Sample test files

## Contributing

Feel free to contribute by adding solutions or improving existing ones.
