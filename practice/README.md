# Practice Folder

This folder contains practice files and utilities for competitive programming.

## Contents

- `debug_template.cpp` - C++ debugging template with utilities for printing various data structures
- `test.cpp` - CSES problem list reference
- `template.py` - Python template with common input/output functions
- `test.py` - Python test file for algorithm testing
- `input.txt`, `output.txt`, `text.txt` - Sample input/output files for testing

## Usage

These files are meant for practicing and testing solutions locally before submitting to online judges.
